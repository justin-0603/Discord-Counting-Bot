# Discord-Counting-Bot
Have you ever seen Counting bot/game on discord? This macro makes you easy to play that game
Discord Counting bot is a mini game bot for servers. One player says a number, and another player says the next number.
This macro is made to do that automaticly.
Requirements:
- Knows How to make secreats
- Have 2 discord accounts

[Instructions]
1. Type pip install -r requirements.txt on terminal
2. Add your Account Tokens in .env
3. type python main.py on terminal
